# Command Center Dashboard

A personal operations dashboard that brings together everything I'm building — Lynn Living (e-commerce), Lynn Care (home care agency), my Lynn Care Foundation which is my NGO in Ghana, and my cloud engineering journey.

## What it does

- **Sales dashboard** — Etsy, eBay, TikTok Shop sales in one place
- **YouTube analytics** — Views, subscribers, growth trends
- **Market research feed** — Reddit scraper pulling trends in home care, NGO funding, small business
- **Project tracker** — All ventures with timelines and progress
- **Document hub** — Google Drive integration for SOPs and contracts

## Tech stack

- Frontend: React, S3, CloudFront
- Backend: AWS Lambda, API Gateway
- Database: DynamoDB
- Infrastructure: Terraform
- Monitoring: CloudWatch
